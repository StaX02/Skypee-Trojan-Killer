# Skypee-Trojan-Killer
经典Skypee摆渡病毒查杀软件

## Descirption
这个程序是我2017年读高中的时候写的了，其实是一个很粗糙的批处理程序，还有一些问题没修。

上传上来权当做个留念，有需要的话也可以拿来用，可用性不保证，建议读完代码再用了。

## Usage
本套批处理程序为清除Skypee病毒专用
分为两个程序，一个是清除U盘中的病毒的，一个是清除主机中的病毒的。
使用方法：阅读完毕关闭本窗口后，将“skypee杀毒（U盘）.bat”拷贝至U盘第一级目录再双击使用。
“skypee杀毒（主机）.bat”拷贝至桌面双击使用（若要在U盘中运行，请右键以管理员方式打开），若未新增程序，请刷新U盘。
切记不要弄错，否则会造成不同程度的损失，甚至使整个电脑崩溃而无法使用！
解锁后，若要拷贝到别的U盘，请复制整个文件夹后，粘贴到另一U盘，再点击“上锁程序.bat”（在U盘中上锁）。
